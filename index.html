Quiz App Using Python Django

Introduction

The Django Quiz App allows users to take quizzes across various subjects, view their results, and track their progress. It features a robust Admin Panel where administrators can add questions, define options, and set correct answers.

Key Features:

  • User Interface: Users select a topic, answer multiple-choice questions, and receive instant scoring.
  • Admin Control: Admins manage questions and answers via the Django admin interface.
  • Data Storage: The Question model stores the question text, four options (A, B, C, D), and the correct answer.
  • Logic: Views handle the scoring logic by comparing user submissions against the database.
  • Educational Use: Ideal for training, interviews, or educational testing. It covers Django fundamentals like dynamic forms, session storage, and model relationships.

Required Modules

To build and run this project, you need the following:

  • Python 3.8+: Ensure Python is installed.
  • Django 3.x or 4.x: The web framework used for the project.
  • pip: Python package manager.

Installation Command:

pip install django

Code Explanation & Source

1. Model Definition (models.py)

This model stores the question, four options, and the correct answer.

from django.db import models

class Question(models.Model):
    question = models.CharField(max_length=500)
    option1 = models.CharField(max_length=200)
    option2 = models.CharField(max_length=200)
    option3 = models.CharField(max_length=200)
    option4 = models.CharField(max_length=200)
    answer = models.CharField(max_length=200)

    def __str__(self):
        return self.question

2. Quiz Logic (views.py)

The index view fetches all questions. When the form is submitted (POST), it checks the selected options against the stored correct answers to calculate the score.

from django.shortcuts import render
from .models import Question

def index(request):
    questions = Question.objects.all()
    if request.method == "POST":
        score = 0
        for q in questions:
            selected = request.POST.get(str(q.id))
            if selected == q.answer:
                score += 1
        return render(request, 'result.html', {'score': score, 'total': len(questions)})
    
    return render(request, 'index.html', {'questions': questions})
